远程登录命令
=======================================

ssh
-------------------------------------------

命令功能
+++++++++++++++
**ssh** 命令用来登录远程设备。

命令格式
+++++++++++++++
**ssh** *ip-address*

参数说明
+++++++++++++++
*ip-address*：设备的IP地址。

命令模式
+++++++++++++++
运维模式

使用指南
+++++++++++++++
无

配置举例
+++++++++++++++
# 登录IP地址为1.1.1.1的远程设备::

 ConnetOS> ssh 1.1.1.1
