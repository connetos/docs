ConnetOS视图类型
+++++++++++++++++++++++++++++++++++++++
ConnetOS提供两个shell视图：

 * Linux shell：Linux界面视图，用户初始登录的视图，亦即Debian Linux系统的缺省shell环境。
 * CLI shell：命令行接口（Command-line Interface）视图，用于对交换机的网络操作和管理。

用户登录到设备后，输入账户和密码，进入到Linux shell，输入cli，即可进入CLI shell。
